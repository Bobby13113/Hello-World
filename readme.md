# 🌍 Hello World Node.js App

This is a simple Node.js "Hello World" application designed to run in a Docker container and be deployed on a Kubernetes cluster. It listens on port **8081** and responds with a plain text message.

---

## 🚀 Features

- Lightweight Node.js HTTP server
- Dockerized for container deployment
- Exposes port 8081
- Ready for Kubernetes deployment

---
